Design Tokens

All design decisions are expressed as named tokens. Reference these in CSS custom properties, Tailwind config, or your framework's theme object. Never hard-code raw hex values.

CSS Custom Properties (Root)

  --ink:        #0c0a07;   /* Primary background — deepest layer */
  --warm-dark:  #131009;   /* Secondary background — alternate sections */
  --mid:        #1e1a12;   /* Tertiary — cards on warm-dark */

  /* Text */
  --parchment:  #e8e0d0;   /* Primary text */
  --text-muted: rgba(232,224,208,0.55);  /* Body / secondary text */
  --text-dim:   rgba(232,224,208,0.30);  /* Labels, metadata */

  /* Accent */

  --gold-dim:   rgba(201,168,76,0.25);   /* Hover states, fills */
  --gold-line:  rgba(201,168,76,0.12);   /* Borders, dividers */

Motion

Timing & Easing

All motion uses a single easing curve — a fast-out-slow-in that feels organic and unhurried. Transitions are either instant micro-interactions (0.2s) or cinematic slow reveals (0.9s). Nothing in between should be animated.

Primary Easing

cubic-bezier(0.16, 1, 0.3, 1)

Used for all scroll reveals and entrance animations. Fast deceleration — content arrives, then settles.

Micro Transition

0.2s ease

Hover states on links, buttons, and nav items. Fast enough to feel responsive, not jarring.

Nav Scroll Transition

0.4s ease

Nav background and padding change on scroll. Leisurely enough to feel intentional.

Scroll Reveal

0.9s cubic-bezier(0.16,1,0.3,1)

opacity 0→1, translateY 30px→0. IntersectionObserver at 12% threshold.

Stagger Delays

0.1s · 0.2s · 0.35s

Applied to sibling elements in the same reveal group. Creates a cascade, not a simultaneous pop.

Glow Pulse Loop

5–8s ease-in-out infinite

Atmospheric background elements only. Slow, barely perceptible breathing effect.

Animations

Three animation patterns used across the site. Copy these keyframes and class patterns directly.

1 — Hero Entrance (CSS animation, fires on load)
@keyframes fade-up {
  from { opacity: 0; transform: translateY(24px); }
  to   { opacity: 1; transform: translateY(0); }
}

/* Apply with staggered delays: */
.hero-eyebrow { animation: fade-up 1s ease 0.3s both; }
.hero-title   { animation: fade-up 1.2s ease 0.5s both; }
.hero-sub     { animation: fade-up 1s ease 0.8s both; }
.hero-ctas    { animation: fade-up 1s ease 1.0s both; }

/* Note: use animation-fill-mode: both (shorthand: 'both')
   so elements start invisible before the delay fires. */
2 — Scroll Reveal (IntersectionObserver + CSS class toggle)
/* CSS */
.reveal {
  opacity: 0;
  transform: translateY(30px);
  transition: opacity 0.9s cubic-bezier(0.16,1,0.3,1),
              transform 0.9s cubic-bezier(0.16,1,0.3,1);
}
.reveal.visible   { opacity: 1; transform: translateY(0); }
.reveal-delay-1   { transition-delay: 0.1s; }
.reveal-delay-2   { transition-delay: 0.2s; }
.reveal-delay-3   { transition-delay: 0.35s; }

/* JS */
const observer = new IntersectionObserver(entries => {
  entries.forEach(e => {
    if (e.isIntersecting) {
      e.target.classList.add('visible');
      observer.unobserve(e.target); /* fire once only */
    }
  });
}, { threshold: 0.12 }); /* 12% visibility triggers reveal */

document.querySelectorAll('.reveal').forEach(el => observer.observe(el));
3 — Atmospheric Glow Pulse (looping background elements)
@keyframes glow-pulse {
  0%, 100% { opacity: 0.5; transform: scale(1); }
  50%       { opacity: 1;   transform: scale(1.05); }
}

/* Used on hero and events background pseudo-elements */
.hero-bg-orb {
  animation: glow-pulse 6s ease-in-out infinite;
}
.events-bg-orb {
  animation: glow-pulse 5s ease-in-out infinite;
}
4 — Marquee Ticker
@keyframes marquee {
  from { transform: translateX(0); }
  to   { transform: translateX(-50%); } /* -50% = moves one copy's width */
}

/* HTML: duplicate content twice inside .marquee-track so it loops
   seamlessly. The animation moves exactly one copy's width. */
.marquee-track {
  display: flex;
  white-space: nowrap;
  animation: marquee 30s linear infinite;
}

Atmospheric Effects

Two global effects applied to the entire page. These are what give the site its analogue warmth and depth — they should be present on every implementation.

1 — Grain / Film Noise Overlay
/* Applied as a fixed pseudo-element on body.
   Creates analogue warmth — like candlelight grain on film. */
body::before {
  content: '';
  position: fixed;
  inset: 0;
  background-image: url("data:image/svg+xml,%3Csvg viewBox='0 0 256 256' xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter id='noise'%3E%3CfeTurbulence type='fractalNoise' baseFrequency='0.9' numOctaves='4' stitchTiles='stitch'/%3E%3C/filter%3E%3Crect width='100%25' height='100%25' filter='url(%23noise)' opacity='1'/%3E%3C/svg%3E");
  opacity: 0.035; /* CRITICAL: keep this subtle — 0.03 to 0.04 max */
  pointer-events: none;
  z-index: 9998; /* sits above content, below cursor */
}
2 — Custom Gold Cursor
/* Hide default cursor */
body { cursor: none; }

/* Gold dot element */
#cursor {
  position: fixed;
  width: 10px; height: 10px;
  background: var(--gold);
  border-radius: 50%;
  pointer-events: none;
  z-index: 9999;
  transform: translate(-50%, -50%);
  transition: width 0.3s ease, height 0.3s ease, opacity 0.3s ease;
}

/* JS: follow mouse + scale on interactive elements */
document.addEventListener('mousemove', e => {
  cursor.style.left = e.clientX + 'px';
  cursor.style.top  = e.clientY + 'px';
});
document.querySelectorAll('a, button').forEach(el => {
  el.addEventListener('mouseenter', () => {
    cursor.style.width   = '20px';
    cursor.style.height  = '20px';
    cursor.style.opacity = '0.5';
  });
  el.addEventListener('mouseleave', () => {
    cursor.style.width   = '10px';
    cursor.style.height  = '10px';
    cursor.style.opacity = '1';
  });
});
3 — Hero Radial Glow Background
/* Applied behind hero image as an overlay.
   Creates warm candlelight atmosphere. */

  background:
    radial-gradient(ellipse at 30% 60%, rgba(201,168,76,0.08) 0%, transparent 60%),
